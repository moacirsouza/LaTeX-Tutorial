# LaTeX-Tutorial, by Moka

Usei o (parco) tempo livre que tenho para seguir um dos tutoriais de LaTeX que
me parece mais divertido, simples e objetivo: o
[LaTeX-Tutorial](https://latex-tutorial.com).

Aqui você acompanha a evolução geral da minha empreitada, seguindo os registros
do que eu estudei a cada dia.

O histórico é mantido em uma tabela, cujo formato é definido pelas **Regras
para Registros de Evolução de Projetos**
([R.R.E.P.](https://github.com/moacirsouza/rrep)).
